Welcome

to the Special Event F1 WebApi

Here you can find information about the drivers and the teams that are participating in the 2023 F1 season, and also who have won some of the races so far.

Get all drivers

You can access the list of drivers by using the /drivers endpoint

http://localhost:5261/api/Drivers

Get singel driver by name

You can also use the drivers list to search drivers by name. /Drivers/GetDriverByName/

http://localhost:5261/api/Drivers/GetDriverByName/max

Filter on grand prix location

The endpoint /GetGrandPrix/ let you see the winner by searching on the location

http://localhost:5261/api/Races/GetGrandPrix/Singapore 

{
    'id': 3,
    'winnerName': 'Carlos Sainz',
    'winnerTime': 1.46,
    {...}
